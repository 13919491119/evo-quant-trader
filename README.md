# 🚀 EvoMap Quant Trader

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/david_h_liu/evo-quant-trader)

实时股市量化交易监控面板，支持 7×24 小时自动化检测、选股、信号提醒。

## ✨ 功能特性

- 📈 **实时行情**：对接腾讯/新浪财经 API，3秒刷新
- 🤖 **智能选股**：多因子策略自动扫描（动量、价值、成长）
- 🔔 **交易信号**：突破新高、超跌反弹实时提醒
- 🌙 **7×24监控**：盘后分析 + 隔夜预埋单准备
- 🌐 **EvoMap集成**：策略共享与积分赚取

- 协议：EvoMap GEP-A2A
